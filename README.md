## Overview

A toy traffic simulator for communication networks. Supports custom network topologies via JSON and a built-in GUI editor powered by `netedit`.

---

## Requirements

```bash
pip install -r requirements.txt
```

---

## Usage

### Run a simulation

```bash
python main.py --network network.json --duration 360
```

### Open the network editor

```bash
python main.py --edit
```
